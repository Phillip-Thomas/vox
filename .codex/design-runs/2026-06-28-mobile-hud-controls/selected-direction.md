# Selected Direction

## Selected Thesis

Suit Telemetry Stack.

## Rejected Directions

- Edge Split Minimal: too raw and not a meaningful health-stat improvement.
- Bottom Command Console: violates the requested top-left vitals placement and would crowd gameplay.

## Why The Winner Was Selected

- It directly addresses the joystick overlap.
- It turns vitals from debug-like decay bars into readable survival telemetry.
- It creates a clear HUD system path: shared glass styles, explicit stack offsets, and a modeled touch action layout.

## Taste Assumptions

- Compact telemetry is better than oversized mobile panels.
- Numeric values help health stats feel more deliberate without adding tutorial copy.
- The primary thumb action should sit exactly in the bottom-right corner.

## Human Review Later

- Whether the top-left vitals panel should be even slimmer after playtesting.
- Whether `USE` should remain the mobile label for `F` or be context-sensitive later.
