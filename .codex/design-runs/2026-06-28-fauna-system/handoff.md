# Handoff

Implement a new fauna system using the flora renderer as the structural template.

## Required Output

- `faunaDensity` and `faunaMaxDistance` graphics-quality knobs.
- `utils/faunaField.ts` with profile, placement, geometry, material uniforms, wind/reality update helpers, and instance builder.
- `components/FaunaField.tsx` with layered instanced meshes per fauna kind.
- Main scene mount near flora/trees.
- `?effects=fauna` voxel test harness.
- Focused utility tests.
- Screenshots and motion proof.

## Selected Visual Language

Low-poly stylized animals with readable silhouettes: grazer, woolly, runner, and hopper. Palette should be climate-aware and slightly alien-tinted when the biome calls for it.
