# Product Brief

Paravox is the public launch name for the existing `vox/main` browser game after merging `origin/collision-efficiency-test` into `master`.

- Page goal: let a visitor load the 3D voxel world at `paravox.com`.
- Primary action: interact with the game canvas.
- User motivation: explore a procedural voxel environment in the browser.
- Success proxy: Firebase-hosted production URL renders the WebGL scene without a framework error overlay.
- Language constraints: use only repo-supported claims; avoid future feature promises.

Required states for this fast launch: initial load, rendered scene, desktop viewport, mobile viewport smoke.
