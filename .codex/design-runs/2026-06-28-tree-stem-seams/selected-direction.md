# Selected Direction

Selected: Direction A, Shared Node Rings.

## Rationale

The issue was not bark color or tone mapping. The trunk mesh was built from independent tube chunks. Sharing skeleton-node rings makes adjacent vertical stem segments use the same vertices, so normals and lighting interpolate through the joint.

## Taste Assumption

Keep the current stylized bark texture and low-poly silhouette; only remove the disconnected chunk look.
