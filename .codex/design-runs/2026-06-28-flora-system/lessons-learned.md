# Lessons Learned

- Mid-story flora density should not use tree-like linear probability. Sparse objects need perceptual density scaling so small debug patches still prove the system.
- Small plant geometry reads better with direct stylized vertex color than PBR unless the forms are large enough for lighting to resolve.
- Cactus/succulent bodies need their own color rule; inheriting canopy hue can make them look like alien silhouettes rather than arid plants.
- Future ecology systems should include both lush and arid harness screenshots, because one seed cannot prove species weighting.
