# Selected Direction

Selected: Direction A, Ecology Layer.

Why: flora should be a visual ecology system, not collectible forage or grass internals. A separate field lets quality, biome, wind, and reality-stage inputs be explicit and reusable.

Rejected:

- Direction B because gameplay pickup density should not control cosmetic world richness.
- Direction C because flowers, fans, cacti, and shrubs need geometry beyond grass blade cards.

Taste assumption: the first pass should be clearly richer but still stylized, not a hyper-detailed botanical simulator.
