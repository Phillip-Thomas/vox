# Product Brief

## Goal

Raise procedural trees from player-scale props into believable canopy-scale forest elements.

## User Motivation

The user wants trees to create a sense of being under and around a living canopy, without making the world feel absurdly oversized.

## Success Proxy

- Tree profile minimum height is materially higher than the previous `3.5` world-unit floor.
- Species span a noticeably broader but capped range.
- In-world trees no longer shrink below the new canopy baseline.
- Visual proof exists in the tree harness and in-world under-canopy shot.

## Language And Claims

No user-facing copy changes. Any documentation must describe deterministic scale, bounded height, and screenshot evidence plainly.
