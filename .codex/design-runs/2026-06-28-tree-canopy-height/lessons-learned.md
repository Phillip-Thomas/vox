# Lessons Learned

- Tree scale should live in `treeProfile`, not as a late render-only multiplier, so the harness and game agree.
- Raising height needs proportional but capped crown and trunk mass scaling.
- In-world instance scale floors matter as much as profile floors when the user is judging minimum tree size.
- The tree-test harness should keep reporting physical dimensions because visual scale issues are otherwise easy to misdiagnose.
