# Selected Direction

Selected: Direction A, Profile-Level Canopy Scale.

## Taste Assumptions

- "Look up at the canopies" means baseline trees should clear the player comfortably.
- "Nothing crazy big" means hard caps around 11 profile units before instance variation.
- Fullness should rise with height so trees do not become tall, sparse sticks.

## Rejected

- Instance-only scaling was rejected because it would hide the real profile problem.
- Giant variants were rejected as too much for this pass.
