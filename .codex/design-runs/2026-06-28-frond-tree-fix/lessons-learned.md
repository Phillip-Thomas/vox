Lessons learned:
- Frond should not use the generic phyllotaxis spray. It needs rib-tangent placement so leaves look attached.
- Prune implementation scaffolding from rendered bark when the skeleton is only a foliage guide.
- Budget leaf sampling against actual emitted cards per species path; otherwise custom emitters can silently underfill.
- Frond screenshots need both front and angled captures because a radial crown can look overly flat from a single camera.
